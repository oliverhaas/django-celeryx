# Dashboard

The metrics dashboard provides an overview of your Celery cluster's performance.

## Metrics

- **Throughput** — Tasks processed per second/minute
- **Latency percentiles** — p50, p95, p99 task execution times
- **Success/failure rates** — Task outcome distribution
- **Per-worker breakdowns** — Metrics grouped by worker
- **Per-task-name breakdowns** — Metrics grouped by task type
- **Time windows** — Last 1m, 5m, 15m, 1h
