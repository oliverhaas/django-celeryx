# Dashboard

The metrics dashboard provides an overview of your Celery cluster's performance.

## Charts

- **Throughput** — Succeeded/failed tasks per time bin (Chart.js)
- **Slowest tasks** — Average runtime with standard deviation by task type
- **Failure rate** — Failure percentage by task type
- **Worker load** — Task counts per worker
- **Top tasks** — Most frequently executed task types

## Time Periods

Filter the dashboard by time period:

- **Today** — Last 24 hours
- **Last 7 days**
- **Last 30 days**

## Filters

- Queue — filter by routing key
- Worker — filter by worker hostname
